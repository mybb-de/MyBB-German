1. Einleitung

Sie haben sich dafür entschieden, MyBulletinBoard als Forensoftware zu verwenden. Wir möchten uns für Ihre Wahl bedanken und hoffen, dass Sie mit der Software zufrieden sind.

Damit möglichst viele Nutzer MyBulletinBoard verwenden können, ist das Forum mehrsprachenfähig. Standardmäßig wird MyBulletinBoard mit englischer Sprache installiert, aber dank vieler Freiwilliger, ist das MyBulletinBoard auch in anderen Sprachen erhältlich. Dieses Sprachpaket erlaubt es Ihnen, MyBulletinBoard auf Deutsch zu betreiben.

2. Installation

Der deutsche Text ist in Dateien gespeichert, die Sie im Ordner "Sprachdateien" finden. Die deutschen Standardbuttons befinden sich im Ordner "Buttons".

Im Folgenden wird in einzelnen Schritten beschrieben, wie Sie vorgehen müssen, um das Sprachpaket zu installieren.
  1. Öffnen Sie Ihr FTP-Programm und stellen Sie eine Verbindung mit Ihrem Server her.

  2. Öffnen Sie den Ordner, in den Sie MyBulletinBoard installiert haben.

  3. Öffnen Sie den Ordner inc und darin den Ordner languages.

  4. Kopieren Sie die Datei deutsch_sie.php und den Ordner deutsch_sie aus dem Ordner "Sprachdateien" in den Ordner languages auf dem Server.

  5. Gehen Sie zurück in den Ordner, in den Sie MyBulletinBoard installiert haben.

  6. Öffnen Sie den Ordner images.

  7. Kopieren Sie den Ordner deutsch_sie aus dem Ordner "Buttons" in den Ordner images auf dem Server.
Sie haben nun alle erforderlichen Dateien auf den Server übertragen. Im folgenden Absatz wird beschrieben, wie Sie Ihr Forum auf das neue Sprachpaket umstellen können.

4. Benutzung

Nachdem Sie alle erforderlichen Dateien auf den Server übertragen haben, können Sie das Sprachpaket für Ihr MyBulletinBoard verwenden. Hier erfahren Sie, wie Sie die deutsche Sprache für das Forum und/oder das Admin Control-Panel einsetzen und wie Sie die deutsche Übersetzung als Standardsprache einstellen. Die Standardsprache wird für Gäste und registrierte Mitglieder verwendet, die keine Sprachauswahl getroffen haben.
  • Um die deutsche Übersetzung im Forum zu benutzen, muss ein Benutzer in seinem Control-Panel unter "Optionen ändern"/"Edit Options" die Sprache "Deutsch" auswählen.

  • Um die deutsche Übersetzung auf den Administrationsbereich anzuwenden, müssen Sie sich im Admin-CP anmelden. Gehen Sie in die "Konfiguration". Unter "Control Panel-Einstellungen"/"Control Panel Preferences (Global)" können Sie die Sprache ändern.

  • Um die deutsche Übersetzung als Standard-Sprache für Ihr Forum zu setzen, müssen Sie sich im Admin-CP anmelden. Gehen Sie in die "Konfiguration". Unter "Allgemeine Konfiguration"/"Global Configuration" können Sie "Deutsch" als Standard-Sprache wählen.
Nach oben

5. Einstellungen

Wenn Sie das Board auf deutsch betreiben, werden im Folgenden Einstellungen erläutert, die angepasst werden sollten. Die Einstellungen finden Sie im Admin-CP unter "Foreneinstellungen" -> "Ändern".

Allgemeine Konfiguration
  • Dezimalpunkt: "," (Komma)

  • Tausendertrennung: "." (Punkt)
Datums- und Zeitformat
  • Datumsformat: "d.m.Y"

  • Zeitformat: "H:i"

  • Zeitzone: "+1"
Desweiteren empfiehlt sich die Übersetzung der Profilfelder ("Benutzer und Gruppen" -> "Profilfelder") und der Benutzergruppen ("Benutzer und Gruppen" -> "Gruppen verwalten").

7. Hinweise

MyBulletinBoard ist ein relativ junges Softwarepaket. Die Unterstützung von mehreren Sprachen existiert erst seit kurzem, deshalb kann die Übersetzung noch kleinere Fehler beinhalten. Wir bitten Sie deshalb, Fehler auf MyBBoard.de zu melden. Ihre Meldungen/Kommentare sind ein wichtiger Bestandteil zur Verbesserung der Übersetzung!

Ein besonderer Dank gilt allen Beta-Testern, die bei der Verbesserung des Sprachpakets geholfen haben!

Wir wünschen Ihnen viel Spaß und Erfolg mit dem MyBulletinBoard!
